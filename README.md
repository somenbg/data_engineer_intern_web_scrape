# data_engineer_intern_web_scrape
> This script scrapes Top USA cities by population from Wikipedia

## Packages required:
  * urllib
  * bs4
  * regex
  
## Installation:

  > pip install bs4
  
## Usage:

  > python top_cities.py\
  > **This will create a csv file containing the city list on the same path**
